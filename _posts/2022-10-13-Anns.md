# Activation Functions

- Activation functions are used to introduce non-linearity in model

- An activation function takes the dot-product of weights and inputs and performs certain computations on it. 
- They activate next layers neurons. 
- We put a certain activation function inside of neurons of hidden layers based on the range of the result we expect to see
- activation functions should be differentiable, because we need this property using backpropagation optimization.
  1. **linear** : Cannot learn complex functions, they are just polynomials of one degree.
  2. **Non-Linear** : More Usable and desirable

# Model Optimization

A neural network F learn by recognizing patterns and try to predicts targets $t^i$ from inputs $x^i$ after being trained on some data. 

So, Ideally we want to have a neural net like

$$
F(x^{(i)}) = t^{(i)}
$$

but generally we consider error $\epsilon_i$ in our function $y^{(i)}=F(x^{(i)})$ so,
$$
t^{(i)} = y^{(i)} + \epsilon^{(i)} \newline
\epsilon^{(i)} = t^{(i)}-y^{(i)}
$$
 $y^{(i)}$ depends on parameters like weights and biases and targets are fixed so its a optimization problem to minimize 
$$
E = \frac{1}{N}\sum_{i=1}^{p}||t^{(i)}-y^{(i)}||^2
$$
Where $N$ is number of training  patterns. So how do we minimize $E$? 



## Backpropagation with Gradient Decent

- To optimize E we use gradient decent algorithm 

- in which we update $Weights$ and $Biases$ in such a way that it reduces error $\epsilon_i$. 

- So we need the direction and a rate to move towards lowering the Cost Function

  **Direction:** the direction is given by gradient of $E$. i-e, Moving in opposite direction to Gradient.

  **Learning Rate** The rate by which we determine how fast we go is called learning Rate $\alpha$ which we define by hit and trial error. Typical values for a neural network with standardized inputs -*(or inputs mapped to the (0,1) interval)* are less than 1 and greater than $10^{-6}$.

 Let we define a generic parameter $u$ (Weight or bias etc). this will be updated in a following way:
$$
u_{new} = u_{old} - \alpha\frac{\partial E }{\partial u} \newline \quad = u_{old} + \alpha\Delta
$$
Where $\alpha$ is learning rate it should not be too small to take a lot of time and not too big to miss out global minima

# General Structure

-  **Single Layer Perceptron** : the simplest feed-forward neural net-
  work with no hidden layers
  
- **Multi Layer Perceptron:** has one or more hidden layers which is
  useful for practical applications. It can learn
  not only linear functions but also non-linear functions. 

  Each neuron has following

  **1. Input:**

  Inputs are the features that are fed into the model for  learning purposes. The input in    Image Classification can be an  array of pixel values pertaining to an image.

  **2. Weights**:

  Weight is a parameter that decides how much **influence** the  input will have on the output. Inputs that are not so important will be  scaled down and important inputs will be scaled up.

  **3. Bias:**

  The role of bias is to control the activation of a node.  Bias can delay the activation, similarly, it can accelerate the  activation.

  **4. Transfer Function**:

  The role of the transfer function is to combine the weighted inputs to output so that the activation function can be applied.

  **5. Activation Function:**

  An Activation decides whether a neuron should be activated  or not. It will activate the neuron if it finds the input important in  the process of predicting using mathematical operations.

Each hidden layer neuron has its own weights and input. Each stores the dot product of corresponding inputs $IN_i$ and weights $W_i$ called as pre-state P or $N_i$ which go through certain activation function  $\sigma$

$$
S = \sigma(Wi \cdot INi + bi)
$$


$$
where \qquad N_i = W_i \cdot IN_i
$$



Where S is state value which encounter next weights matrix and serves as input for next layer
